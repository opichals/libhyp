Use highlights:

  • NEWS!!! - browse .hyp files inside a software archive directly by using the archive URL (LHA/ZIP/TGZ/TBZ2)
  • browse through your own .hyp collection exposed via FTP or HTTP server
  • no initial to HTML conversion overhead (everything happens on the fly)

HYP publishing recomendations:

  • expose bare .hyp file as is on site so that it is immediatelly browsable without any .zip extractions etc. (.hyp already uses lha compression).
  • perhaps you can add a direct link to libhyp.atari.org clicking on which anyone would be able to browse the hypertext.

Here is a couple of direct .hyp browse file link examples:

  • Direct .hyp file: http://libhyp.atari.org/hypview.cgi?url=http://jaysoft.atari.org/docs/ataripf.hyp
  • Inside an Lharc archive: http://libhyp.atari.org/hypview.cgi?url=http://jaysoft.atari.org/docs/dspdoc.lzh
